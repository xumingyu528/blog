示例代码

#include "apue.h"

int 
main(void)
{
	printf("hello world from process ID %ld\n", (long)getpid());
	exit(0);
}

解释

  • getpid() 函数可以获取当前进程ID

  • 大多数进程ID可以用整型表示，这里强制转换为 long 是为了提高可移植性
